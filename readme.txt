Project Description

----------------

Diploma project on the topic "Recreation of sounds using a genetic algorithm and additive synthesis"

----------------

Usage

----------------

Argument #1: Input Filename
Argument #2: Output Filename
Argument #3 (Optional): Generation Count
Argument #4 (Optional): Population Count

----------------

Examples

----------------

Windows: run.bat input.wav output.wav 20 80
Unix: $PWD/run.sh input.wav output.wav 20 80
